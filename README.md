خطوات العمل : 
1-افتح فولدر الـ DataBase  المرافق .
2-افتح سكريبت انشاء الداتا بيز (المرقم بالرقم 1) وقم بتنفيذه في smss  لانشاء الداتا بيز.
3-نفذ باقي الـ scripts  بنفس الترتيب المرقمين به (الترتيب في غاية الأهمية ) .  
5-قم بتشغيل المشروع من خلال الملف ModernResturantSystem.sln. 
6-عند تشغيل الملف أول مرة ضع اسم المستخدم "Admin" وكلمة المرور "Admin"
7-سوف تفتح معك الفورم الخاصة باعدادات الاتصال بقاعدة البيانات ضع اسم قاعدة البيانات ResturantMDB وباقي البيانات حسب اعدادات الاتصال بالسيرفر وقم بالحفظ. 
8-سيقوم البرنامج بفحص الاتصال قبل الحفظ ولن يسمح ببيانات اتصال غير صحيحة . 
9-بعد الحفظ ستفتح فورم تسجيل الدخول مرة أخري سجل باسم المستخدم"TESTUSER" وكلمة المرور "TESTPASSWORD". 
11-يمكن إضافة وتعديل المستخدمين من داخل التطبيق بعد ذلك وتحديد الصلاحيات واضافة الاطباق.
10-انتهت العملية والبرنامج يعمل .
